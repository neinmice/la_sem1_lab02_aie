# Лабораторная работа: Форматы хранения разреженных матриц

## Описание
В данной лабораторной работе требуется реализовать базовые операции над разреженными матрицами
в форматах **COO**, **CSR** и **CSC**.  
Реализация должна строго соответствовать заданным сигнатурам классов.

Проверка корректности решений осуществляется автоматически с помощью **GitHub Actions**.

---

## Порядок выполнения

* Описан в разделе "Практическая часть" условия лабораторной

---

## Автоматическая проверка

* Проверка выполняется с помощью **GitHub Actions**.
* Тесты запускаются автоматически при создании Pull Request.
* Эталонное решение и тесты скрыты.
* Результаты проверки отображаются на странице Pull Request.

Если тесты временно недоступны, Pull Request будет автоматически отклонён с соответствующим сообщением.

---

## Ограничения

* Описаны в разделе "Правила выполнения работы" условия лабораторной

---

## Требования к коду

* Код должен быть читаемым и структурированным
* Запрещено изменять сигнатуры классов и методов
* Решение должно корректно обрабатывать граничные случаи
